# 投資風險免責聲明 (Investment Disclaimer)

## 中文

- 本軟體的任何輸出（包含但不限於績效指標、統計裁決、策略骨架、回測結果、
  產生的交易程式）**不構成任何形式的投資建議、財務建議或交易推薦**。
- 本軟體的作者與貢獻者**並非持牌的投資顧問或財務顧問**。
- **投資有風險，盈虧自負。過去績效不代表未來表現。**
- 統計檢定能降低被運氣誤導的機率，但**無法保證任何策略未來會獲利**。
  「具統計優勢」的裁決只代表現有樣本下的證據，不是對未來的承諾。
- 使用者依本軟體做出的任何投資決策及其後果，**概由使用者自行承擔**。
- 本軟體可協助產生連接券商 API 的程式框架，但：
  - 預設一律為紙上模擬，不會自動連接真實帳戶下單。
  - 真實下單需使用者自行填入金鑰、自行解除安全閘門、自行承擔全部風險。
  - 作者與貢獻者不對任何因真實下單而產生的損失負責。
- 作者與貢獻者**不對任何因使用本軟體而產生的直接或間接損失負責**。

如果你正在虧損、借錢投資、或無法控制交易行為，請尋求專業協助。
