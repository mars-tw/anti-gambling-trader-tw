# 反詐指南 — 拆穿台灣常見的投資詐騙

> 這個工具叫「反詐投資王」，不是叫假的。
> 它存在的真正原因，就是台灣到處都是**假飆股群、假二群、假 VIP 群、假名師、
> 假績效截圖、保證獲利話術、誊騙幣與假投資平台**。
> 這份文件教你辨識它們，並用**統計與數學**戳破它們的承諾。

詐騙的共同核心只有一個：**用「倖存者偏差」與「精選證據」，讓你誤以為有穩賺的捷徑。**
而它們最怕的，就是你冷靜地把它的承諾丟進數學裡檢驗。

想快速自我檢測，直接執行：

```bash
anti-gambling-trader scam-check
```

## 一、假飆股群 / 假 VIP 群 / 假二群

### 長什麼樣
- 有人**主動**把你拉進 LINE / Telegram 群，群裡「老師」每天報明牌。
- 群裡一堆「學員」貼獲利截圖、感謝老師，氣氛一片祥和。
- 先給你幾支「會漲」的股票嘗甜頭，再叫你**升級 VIP / 進二群**付費。
- 要你下載指定 App、加指定客服、或匯款到「老師合作的平台」。
- 禁止你公開質疑，提問會被踢或私訊安撫。

### 為什麼是詐騙
那些「學員」與截圖多半是樁腳或假帳號 —— **倖存者偏差的人造版**。
老師同時對不同群報**相反**的方向，總有一群會猜中；猜中的被收割成付費會員，
猜錯的被踢出。你看到的「神準」，是被刻意製造出來的。

### 怎麼拆穿
> 真的能穩定預測股市的人，會自己默默賺，不會免費拉你進群。

把「老師」的歷史推薦**全部**記下來（含失敗的），用本工具算期望值 ——
幾乎沒有例外，都是負的。

## 二、保證獲利 / 高勝率話術

### 長什麼樣
- 宣稱「保證獲利」「穩賺不賠」「零風險高報酬」。
- 強調「勝率 90% 以上」，卻從不提**賠的時候賠多少**。
- 用「月配 X%」包裝成像定存一樣安全。
- 催你「現在不進場就錯過」，製造急迫感。

### 為什麼是詐騙
高報酬必然伴隨高風險，這是金融鐵律 —— 任何「高報酬 + 零風險」不是騙就是龐氏。
而「高勝率」本身會騙人：**勝率 90% 但賺小賠大，一次大賠就回吐所有獲利，期望值仍是負的。**

### 怎麼拆穿
> 如果穩賺，他為什麼要找你的錢，而不是去跟銀行無限借錢自己賺？

本工具的統計檢定，就是要拆穿「高勝率 = 賺錢」的幻覺 —— **看的是期望值，不是勝率。**

## 三、假名師 / 假績效截圖

### 長什麼樣
- 炫耀豪車、名錶、出國，暗示「跟我就能財富自由」。
- 貼「對帳單 / 獲利截圖」，但截圖可以 P、可以只挑賺的那幾天。
- 大量「學員見證」，卻沒有可驗證的長期、完整、第三方紀錄。
- 課程 / 訂閱費很貴，且強調「名額有限」。

### 為什麼是詐騙
截圖是**倖存者偏差**的完美載體：他只給你看賺的那幾筆，不給你看賠光退場的。
就算對帳單是真的，幾天的暴賺也無法證明任何優勢 —— 那可能只是運氣。
很多「名師」的真正收入來自學費與抽佣，不是交易本身。

### 怎麼拆穿
> 要求看「完整、連續、夠長（數百筆以上）」的交易紀錄，而不是精選截圖。

把那份紀錄丟進本工具做**顯著性檢定**與**樣本外驗證** —— 真本事過得了，運氣過不了。

## 四、誊騙幣 / 假投資平台

### 長什麼樣
- 叫你把錢匯到某個「平台 / 錢包 / 交易所」，而不是你自己掌控的帳戶。
- 一開始能小額出金（誘餌），加碼後就「系統維護 / 要繳稅 / 要刷流水才能出金」。
- 網站 / App 很像知名交易所，但網址、下載來源不對。
- 宣稱「搬磚套利 / DeFi 高收益」穩賺，要你儲值參與。

### 為什麼是詐騙
錢一旦進了「他們控制的平台」，帳面數字是他們隨便顯示的，你看到的「獲利」是假的。
能小額出金只是釣餌，加碼後就用各種理由卡住出金 —— 這是典型的**殺豬盤**。

### 怎麼拆穿
> 只用「你自己掌控私鑰 / 自己開戶」的正規管道，絕不把錢匯到別人叫你匯的平台。
> 任何「要先繳錢才能領回你的錢」的說法，100% 是詐騙。

## 通用守則

1. **主動找上門的「機會」，先預設是詐騙。** 真正的好機會不需要拉你、催你、保證你。
2. **看完整紀錄，不看精選截圖。** 倖存者偏差是所有投資詐騙的共同燃料。
3. **用期望值，不用勝率。** 高勝率會騙人；負期望值的策略，玩越久賠越多。
4. **錢只放自己掌控的帳戶。** 任何要你匯到「他指定平台」的，高度可疑。
5. **要先繳錢才能領回錢 = 100% 詐騙。** 沒有例外。
6. **慢慢來，別賭。** 真正的優勢經得起統計檢定與時間考驗。

## 遇到詐騙怎麼辦

- 立刻停止匯款、停止「升級」、停止下載對方指定的 App。
- 保留所有對話與匯款紀錄。
- 台灣可撥打 **165 反詐騙諮詢專線** 查證與求助。
- 若已受害，盡快報案並聯繫銀行嘗試圈存款項。
