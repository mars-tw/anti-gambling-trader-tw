# 新手快速上手指南（完全沒用過電腦命令也能跟著做）

這份指南是給**完全的新手**看的 —— 就算你沒聽過「終端機」、沒裝過 Python、
沒用過 AI，只要照著一步一步做，大約 **15 分鐘**就能用這個工具分析你的交易、
看出自己到底是在投資還是在賭博、甚至檢查自己是不是遇到投資詐騙。

> 看不懂某個詞沒關係，照著打就對了。每一步都會告訴你「應該會看到什麼」，
> 對得上就代表你做對了。

## 這個工具能幫你做什麼？

- 📊 把你的交易紀錄（買賣股票 / 加密貨幣的紀錄）丟進去，告訴你**真的有賺還是其實在賠**
- 🎲 用數學判斷你是「有方法」還是「運氣好 / 在賭博」
- 🛡️ 檢查你是不是遇到**假飆股群、假老師、保證獲利、假投資平台**等詐騙

## 第一步：安裝 Python（這個工具的引擎）

這個工具是用一種叫 **Python** 的程式語言寫的，所以要先把 Python 裝到你的電腦上。
**只需要裝一次**，以後就不用再裝了。

### 🪟 如果你用 Windows

1. 打開你的網頁瀏覽器（Chrome、Edge 都可以）。
2. 在網址列輸入 **python.org/downloads** 後按 Enter。
3. 你會看到一個黃色的大按鈕，寫著 **「Download Python 3.x.x」**（x 是數字），點它。
4. 下載完成後，打開那個檔案（通常在「下載」資料夾，檔名像 `python-3.x.x.exe`）。
5. **⚠️ 非常重要**：安裝畫面最下方有一個小框框
   **「Add Python to PATH」**（把 Python 加入 PATH），**一定要打勾！**
   （沒勾的話後面會找不到 Python。）
6. 然後點 **「Install Now」**，等它跑完，看到「Setup was successful」就完成了。

### 🍎 如果你用 Mac

1. 打開瀏覽器（Safari、Chrome 都可以）。
2. 在網址列輸入 **python.org/downloads** 後按 Enter。
3. 點黃色大按鈕 **「Download Python 3.x.x」**。
4. 下載完成後，打開那個檔案（檔名像 `python-3.x.x.pkg`）。
5. 一直點「繼續」→「同意」→「安裝」，輸入你開機的密碼，等它跑完即可。

## 第二步：打開「終端機」（你輸入指令的地方）

「終端機」是一個可以打字下指令給電腦的黑色（或白色）視窗。聽起來很難，
其實你只要會**複製、貼上、按 Enter** 就夠了。

### 🪟 Windows 打開終端機

1. 按鍵盤左下角的 **田字鍵（Windows 鍵）**，或點螢幕左下角的開始圖示。
2. 直接打字輸入 **PowerShell**。
3. 看到 **「Windows PowerShell」** 出現，點它打開。
4. 會跳出一個藍色或黑色的視窗，游標在閃 —— 這就是終端機，準備好了。

### 🍎 Mac 打開終端機

1. 按鍵盤的 **Command（⌘）+ 空白鍵**，會跳出搜尋框（Spotlight）。
2. 輸入 **Terminal**（終端機）。
3. 按 Enter，會打開一個白色或黑色的視窗 —— 這就是終端機。

### ✅ 確認 Python 裝好了

在終端機視窗裡，**複製貼上**下面這行，然後按 Enter：

```
python --version
```

> Mac 使用者如果出現錯誤，改打 `python3 --version` 試試。

如果看到類似 **`Python 3.10.x`**（或更大的數字）就代表裝對了！
如果出現「找不到」之類的錯誤，回到第一步重裝，**記得把「Add Python to PATH」打勾**。

## 第三步：下載這個工具

### 方法 A：用 Git 下載（推薦，方便日後更新）

如果你的電腦有「Git」，在終端機貼上這行按 Enter：

```
git clone https://github.com/mars-tw/anti-gambling-trader-tw.git
```

（如果出現「git 不是內部或外部命令」，代表沒裝 Git，請改用方法 B。）

### 方法 B：直接下載 ZIP（最簡單，不用懂 Git）

1. 用瀏覽器打開：**https://github.com/mars-tw/anti-gambling-trader-tw**
2. 找到綠色的 **「Code」** 按鈕，點它。
3. 點下拉選單最下方的 **「Download ZIP」**。
4. 下載完成後，到「下載」資料夾把那個 ZIP 檔**解壓縮**（在檔案上按右鍵 →
   Windows 選「解壓縮全部」、Mac 直接雙擊）。
5. 記住解壓縮後的資料夾位置（例如桌面）。

## 第四步：進入工具的資料夾並安裝

### 1. 進入資料夾

我們要讓終端機「走進」剛下載的資料夾。在終端機打 `cd `（**cd 後面有一個空格**），
然後把資料夾**直接拖進終端機視窗**（拖曳那個資料夾到黑視窗上放開），
它會自動填上路徑，最後按 Enter。

例如看起來會像這樣（你的路徑會不一樣）：

```
cd C:\Users\你的名字\Desktop\anti-gambling-trader-tw
```

> 小技巧：拖曳資料夾到終端機是最不會打錯字的方法。

### 2. 安裝工具

進到資料夾後，貼上這行按 Enter（**Mac 若 `python` 沒反應就改 `python3`**）：

```
python -m pip install -e .
```

等它跑個幾十秒，看到一堆文字跑完、最後沒有紅色錯誤，就裝好了。

## 第五步：開始使用！（最重要的部分）

### 🎬 先看個示範（不用準備任何資料）

直接貼上這行按 Enter：

```
python -m core.cli demo
```

你會看到一份完整的中文分析報告 —— 這是一個「賭博型」交易的範例，
工具會告訴你它為什麼判定這是賭博。**這就是工具的效果**。

想看「有賺錢、有方法」的對照範例，打：

```
python -m core.cli demo --edge
```

### 📝 分析你自己的交易

#### 步驟 1：產生一個空白表格範本

```
python -m core.cli init-template
```

這會在資料夾裡產生一個叫 **`trades_template.csv`** 的檔案。

#### 步驟 2：填入你的交易

用 **Excel** 或**記事本**打開 `trades_template.csv`，你會看到欄位：

| 代號 | 方向 | 進場時間 | 出場時間 | 進場價 | 出場價 | 數量 | 手續費 | 策略 |
|------|------|----------|----------|--------|--------|------|--------|------|

把裡面的範例換成你自己的交易，一行 = 一筆「已經買賣完成」的交易。例如：

| 2330 | 買 | 2025-01-03 | 2025-02-10 | 1000 | 1080 | 1000 | （留空） | 季線突破 |

- **代號**：股票代號（台股像 2330）或幣別（像 BTCUSDT）。
- **方向**：填「買」或「賣」。
- **手續費**：不知道就留空，工具會自動估算。
- **策略**：填你「為什麼買」的理由（例如「聽老師明牌」「自己看技術線」）。
  **強烈建議認真填** —— 這樣工具才能告訴你「哪一招在送錢」。
- 開頭有 `#` 的那幾行是說明，可以留著（分析時會自動跳過）。

填好後**存檔**（如果是 Excel，存檔時選「CSV」格式）。

#### 步驟 3：分析它

```
python -m core.cli analyze trades_template.csv
```

（如果你把檔名改了，就把上面的 `trades_template.csv` 換成你的檔名。
檔名有空格的話，用拖曳的方式把檔案拖進終端機最保險。）

按 Enter，幾秒後就會出現你的**專屬分析報告**：你是優勢還是賭博、
哪一招在送錢、勝率要到多少才會轉正、有沒有詐騙嫌疑……全部都有。

### 🛡️ 檢查自己是不是遇到詐騙

如果你被拉進什麼飆股群、有人保證獲利、叫你升級 VIP 或匯款到某平台，
打這行做個自我檢測：

```
python -m core.cli scam-check
```

它會問你幾個是非題（打 `y` 是 / `n` 否，按 Enter），最後告訴你詐騙風險高不高、
該怎麼自保。

## 常見問題

**Q：打 `python` 沒反應 / 說找不到？**
A：Mac 改打 `python3`。Windows 的話，多半是裝 Python 時忘了勾
「Add Python to PATH」，重裝一次並勾起來。

**Q：中文變成亂碼？**
A：在指令最前面加 `PYTHONUTF8=1`，例如
`PYTHONUTF8=1 python -m core.cli demo`。

**Q：它會告訴我「買什麼股票會漲」嗎？**
A：**不會，而且這正是重點。** 任何宣稱能報明牌、保證獲利的，
都該高度懷疑是詐騙。這個工具只做一件事：用數學誠實告訴你過去的成績，
幫你別再被「穩賺」的話術騙。

**Q：我的交易資料會被傳到網路上嗎？**
A：不會。所有分析都在你自己的電腦上跑，資料不會外傳。
